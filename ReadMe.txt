this program reads a file, skips bad tokens, and counts valid words (letters, hyphens, apostrophes) using a bst.

test input:
hello@world 11word my$money don't-shout. niaz computer mouse keyboard niaz keyboard! mouse, computer?

output:
don't-shout : 1
niaz : 2
computer : 2
mouse : 2
keyboard : 2

total unique words: 5
total words: 9
